ShiftLog

This was a piece of software I wrote to automate making shift logs. These shift logs were specific to my employer at the time. This was a personal program, and not for use by the company. It was made solely for the purpose of making my life easier. It produced a locally saved .xlsx file that I later email to the supervisor. The program was written in C++, and used the OpenXLSX library. The code has long since been lost, but I intend to re-code it someday for the nostalgia.
